# 小白AI工坊 — 品宣部专用功能规划

## 🎯 定位
个人品牌站 + 品宣部内部工作站，归志豪所有，但怡彤/单思维/雨诺/戴腾霄每天都在上面干活。

## 📋 品宣部专属功能

### 1. 📅 内容排期看板
- 日历视图：哪天发小红书、哪天发抖音
- 每人一列：怡彤 | 单思维 | 雨诺 | 戴腾霄
- 拖拽改状态：待做 → 进行中 → 已完成
- localStorage存数据，每周自动存档

### 2. 🎰 AI灵感轮盘
- 一键随机出选题
- 维度：产品线 + 平台 + 内容形式 + 情绪
- 比如：环氧彩砂 × 抖音 × 施工对比 × 反差
- 点一下一个组合，适合没灵感的时候玩

### 3. 📝 AI批量内容工坊
- 一个产品输入（比如：深灰色美缝剂、环氧彩砂）
- 一键生成6个版本：
  - 小红书种草文
  - 抖音口播脚本  
  - 朋友圈短文案
  - 公众号推文大纲
  - 产品卖点海报文案
  - 视频标题+封面文案
- 怡彤和雨诺直接取用修改

### 4. 🏆 内容评分器
- 把写好的文案粘进去
- AI分析爆款潜力（1-10分）
- 给出优化建议
- 同一条内容改完再测，看着分数涨

### 5. 🗂️ 共享素材库
- 4个人都能上传图片/链接
- 分类：产品图 / 施工图 / 表情包 / 竞品截图
- 搜索+标签筛选
- 不用在微信群里翻聊天记录了

### 6. 📊 内容效果追踪
- 简单记录每篇数据的阅读/点赞/评论
- 自动出周报：哪篇爆了、哪个方向该多做
- 志豪可以看全团队产出统计

### 7. 🔥 热点日历
- 建材行业全年营销节点
- 当月高亮：618/双11/装修旺季/施工季节
- 点击节点 → AI生成该节点的选题建议

### 8. 🎯 今日待办
- 每个人登录后（按名字选身份）
- 看到今天该做的事
- 做完打勾，志豪能看到完成率
